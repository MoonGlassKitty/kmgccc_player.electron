## BUG 记录

```text
BUG 名字：播放器 Dock 进度条悬停缺少明显视觉提示
发生的系统：Windows
BUG 的大致类型：界面 / 播放
复现方法：在播放器底部 Dock 区域拖动播放进度条；进度条拖动位置位于底部，但悬停时缺少明显的视觉提示,用户无法知道这里可以拖动进度条。
BUG 的状态：已复现
```

```text
BUG 名字：全屏播放 Dock 自动隐藏延迟和分块悬停判定不同步
发生的系统：Windows
BUG 的大致类型：界面 / 播放
复现方法：进入全屏播放后，如果鼠标不悬停在 Dock，Dock 会延迟一段时间后消失，但当前消失延迟偏短，需要加长延迟时间，或增加设置选项让用户更改延迟消失时间。底部 Dock 被分为两块，鼠标悬停在某一块时另一块会消失，应该改成两块同步消失或同步出现。鼠标悬停在横贯底部的共同判定区域时，两块 Dock 会同时出现和消失；但每一块 Dock 又有自己的悬停判定区域，并且叠在共同的横贯判定区域上，导致鼠标悬停到其中一块 Dock 后另一块 Dock 消失。
BUG 的状态：已复现
```
